# Taxi Driver - Game - Java - 2017
### Simulate Taxi rides in the city.
 
## Features:
- **Game Logic**
	- **Start, Pause, or Stop a Taxi**
	- **Ask for the bill**
	- **Take a free ride**
	- **Calculate the Time of a ride**
	- **Calculate the Price of a ride**
- **ASCII graphic in terminal**
- **Menu navigation**

## Technologies:
- JAVA

## Tools:
- IntelliJ IDEA
